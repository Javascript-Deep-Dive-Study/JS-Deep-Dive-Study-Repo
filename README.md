포스팅 규칙
포스팅을 하는 이유는 공부한 내용을 글로 정리하며 복습하고 개념을 탄탄히 다지기 위함입니다.
따라서 포스팅은 각자의 스타일대로 진행합니다.
스터디원들이 함께 보고 개념을 복습할 수 있도록 좋은 정보들을 자유롭게 올려주시기 바랍니다 ☺️
